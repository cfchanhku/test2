# Flask Grid App

<!-- Project documentation for setup and running the Flask floor-grid application. -->

This project includes a simple Flask web application.

## Requirements

- Python 3.8+

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python3 app.py
```

Then open http://127.0.0.1:5000 in your browser.# test1
